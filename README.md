# Demo eCommerce Platform Overview

A premium, high-performance eCommerce landing page and platform demonstration built with modern web technologies. This project showcases advanced scroll-triggered animations, multi-language support (English/Arabic), and a sleek, responsive design.

## üöÄ Key Features

- **Dynamic Scroll Animations**: Powered by **GSAP** and **ScrollTrigger** for a premium, interactive user experience.
- **RTL & LTR Support**: Fully localized for both English and Arabic speakers, including layout flipping.
- **Modern Tech Stack**: Built with React, TypeScript, and Vite for lightning-fast performance.
- **Rich User Interface**: Custom components styled with Tailwind CSS and Radix UI primitives.
- **Full eCommerce Pages**: Includes Home, Product Detail, Cart, Checkout, Account, and Admin Dashboard.

## üì¶ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/irshad9747/ufuq.git
   cd ufuq
   ```

2. Navigate to the application directory:
   ```bash
   cd app
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

The application will be available at `http://localhost:5173` (or the next available port).
